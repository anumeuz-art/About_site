## 📂 Project Structure
```text

├── index.html       # Landing page (Home)
```

## 🚀 How to Use
1. **Clone the repository:**
   ```bash

   ```
2. **Customize Content:**
   Edit `data.js` to update projects, skills, or personal info.
3. **Launch:**
   Simply open `index.html` in your browser or deploy via GitHub Pages.
